# Memory — devops-engineer

Vùng bộ nhớ & ký ức của vai trò **devops-engineer**. Agent sẽ tham chiếu các file trong đây khi thực thi tác vụ.

## Cấu trúc

| Thư mục | Lưu trữ gì | Ví dụ |
|---------|-----------|-------|
| `discussions/` | Tóm tắt các cuộc thảo luận, meeting notes | `2026-03-26_chien-luoc-q2.md` |
| `decisions/` | Các quyết định đã được phê duyệt | `ADR-001_chon-crm.md` |
| `deliverables/` | Sản phẩm đã bàn giao (bài đăng, báo cáo, hợp đồng...) | `bai-dang-facebook-tinh-nang-moi.md` |
| `data/` | Dữ liệu, số liệu, bảng tính tham khảo | `salary-benchmark-2026.csv` |

## Quy tắc
- Đặt tên file theo format: `YYYY-MM-DD_ten-noi-dung.md`
- Mỗi file nên có tiêu đề rõ ràng và ngày tạo
- Agent sẽ TỰ ĐỘNG lưu các kết quả thảo luận quan trọng vào đây
