# Bộ nhớ Chung — Phòng marketing-sales

Đây là vùng bộ nhớ **chung** của phòng ban. Lưu trữ thông tin liên quan đến toàn bộ phòng ban.

## Cấu trúc
- `decisions/` — Các quyết định quan trọng của phòng ban
- `reports/` — Báo cáo định kỳ (tuần/tháng/quý)
- `data/` — Dữ liệu, số liệu dùng chung

## Cách sử dụng
Agent sẽ tự động tham chiếu các file trong thư mục này khi xử lý tác vụ liên quan đến phòng ban.
