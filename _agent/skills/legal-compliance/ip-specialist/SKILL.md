### IP Asset Inventory for Tech Company

|---------|-----------|-------------------|---------|

### Trademark Registration (Vietnam)
```
Process:
├── Search: Tra cứu nhãn hiệu tại NOIP (1-2 tuần)
├── File: Nộp đơn đăng ký (Classes 9, 35, 42 cho SaaS)
├── Formality Exam: 1-2 tháng
├── Substantive Exam: 6-12 tháng
├── Publication: 2 tháng (opposition period)
├── Grant: Certificate issued → Valid 10 năm, renewable
└── Cost: ~3-5 triệu VNĐ/class (qua đại diện SHTT)

Tips:
├── File EARLY — first-to-file system tại VN
├── File cho multiple classes (9: software, 35: advertising, 42: SaaS)
├── Monitor: Set up watch service for similar marks
└── Enforce: Send cease & desist within 30 days of discovering infringement
```

### Open Source Compliance
| License | Copyleft? | Can Use in Commercial SaaS? | Key Obligation |
|---------|----------|---------------------------|---------------|
| MIT | No | ✅ Yes | Include license text |
| Apache 2.0 | No | ✅ Yes | Include license + NOTICE |
| BSD 2/3 | No | ✅ Yes | Include license text |
| LGPL | Weak | ✅ Yes (if dynamically linked) | Share modifications to LGPL code |
| GPL v2/v3 | Strong | ⚠️ Careful | Must share ALL combined source code |
| AGPL | Strong | ❌ Risky for SaaS | Must share source if accessible over network |
