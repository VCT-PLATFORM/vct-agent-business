# SOP-702: Compliance Audit — Kiểm tra Tuân thủ

> **Actor**: Compliance Officer + General Counsel | **Frequency**: Quarterly

## Quarterly Compliance Checklist
```
// turbo
├── Data Privacy (NĐ 13/2023):
│   ├── [ ] Consent mechanisms functioning
│   ├── [ ] Privacy policy up to date
│   ├── [ ] Data processing activities logged
│   ├── [ ] Cross-border transfer assessment current
│   ├── [ ] Data retention schedule followed
│   └── [ ] Subject access requests processed (<30 days)
├── Labor Law (BLLĐ 2019):
│   ├── [ ] All employees have valid HĐLĐ
│   ├── [ ] BHXH/BHYT/BHTN contributions current
│   ├── [ ] Internal regulations registered (if >10 employees)
│   ├── [ ] Overtime within legal limits
│   └── [ ] Leave entitlements tracked
├── Tax Compliance:
│   ├── [ ] All tax filings on time
│   ├── [ ] Invoice issuance compliant
│   └── [ ] Transfer pricing documented (if applicable)
├── IP Protection:
│   ├── [ ] Trademarks registered/renewed
│   ├── [ ] Open source licenses compliant
│   └── [ ] Employee IP assignment agreements current
├── Cybersecurity (Luật ANMG 2018):
│   ├── [ ] Data localization requirements met (if applicable)
│   ├── [ ] Security incident reporting procedures ready
│   └── [ ] Annual security assessment completed
```

## Output
```
// turbo
├── Compliance Score: [0-100%] per category
├── Findings: Non-compliant items + severity
├── Action Plan: Fix + Owner + Deadline
├── Report to: CEO + Board (quarterly)
├── Lưu vào: legal-compliance/compliance-officer/memory/reports/
```
