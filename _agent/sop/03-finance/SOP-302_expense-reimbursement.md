# SOP-302: Expense & Reimbursement — Tạm ứng & Hoàn ứng

> **Actor**: Employee + Manager + Accounting Manager + CFO

## Quy trình
```
1. REQUEST: Nhân viên tạo yêu cầu (mục đích, số tiền, hạng mục)
2. APPROVE: Manager → CFO (nếu >5M VNĐ)
3. ADVANCE: Accounting tạm ứng nếu cần
4. SPEND: Nhân viên chi tiêu, giữ hóa đơn/receipt
5. REPORT: Nộp báo cáo chi tiêu (trong 7 ngày)
6. VERIFY: Accounting kiểm tra chứng từ
7. SETTLE: Hoàn ứng chênh lệch (trong 15 ngày)
```

## Approval Matrix
| Số tiền | Approver |
|---------|---------|
| <2M VNĐ | Manager trực tiếp |
| 2-10M VNĐ | Manager + CFO |
| >10M VNĐ | Manager + CFO + CEO |

## Chứng từ Yêu cầu
- Hóa đơn VAT (GTGT) cho chi phí >200K VNĐ
- Receipt/Biên nhận cho chi phí <200K VNĐ
- Boarding pass + Vé (công tác)
- E-receipt (dịch vụ online)
