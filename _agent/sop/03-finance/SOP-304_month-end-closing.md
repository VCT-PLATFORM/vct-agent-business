# SOP-304: Month-End Closing — Khóa sổ Cuối tháng

> **Actor**: Accounting Manager + CFO | **Deadline**: Ngày 5 tháng sau

## Checklist Khóa sổ

### Ngày 1-2: Thu thập & Đối chiếu
```
// turbo
├── [ ] Bank reconciliation (đối chiếu ngân hàng)
├── [ ] Accounts Receivable (công nợ phải thu — confirm payments)
├── [ ] Accounts Payable (công nợ phải trả — confirm invoices)
├── [ ] Payroll finalized (lương đã tính — HR Ops confirm)
├── [ ] Expense reports collected (tạm ứng hoàn ứng)
├── [ ] Revenue recognition (deferred → recognized)
```

### Ngày 3: Adjustments
```
// turbo
├── [ ] Accruals (chi phí trích trước: lương, thuê, dịch vụ)
├── [ ] Prepaid expenses (chi phí trả trước: insurance, licenses)
├── [ ] Depreciation (khấu hao TSCĐ)
├── [ ] Foreign currency revaluation (nếu có)
├── [ ] Intercompany transactions (nếu có)
```

### Ngày 4: Reports
```
// turbo
├── [ ] P&L (Báo cáo Kết quả Kinh doanh)
├── [ ] Balance Sheet (Bảng cân đối Kế toán)
├── [ ] Cash Flow Statement (Báo cáo Lưu chuyển Tiền tệ)
├── [ ] Budget vs Actual variance report
├── [ ] SaaS metrics update (MRR, Churn, ARPU)
```

### Ngày 5: Review & Close
```
// turbo
├── [ ] CFO review & sign-off
├── [ ] Lock accounting period (không chỉnh sửa retroactive)
├── [ ] Archive supporting documents
├── [ ] Cập nhật COMPANY_CONTEXT.md (KPIs section)
├── [ ] Monthly report → CEO (dùng template monthly-report.md)
```
