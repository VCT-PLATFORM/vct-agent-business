# SOP-209: Customer Referral — Chương trình Giới thiệu

> **Actor**: CSM + Marketing Manager | **Trigger**: Xây dựng hoặc vận hành referral program

## Program Design
```
// turbo
├── Who can refer: Existing customers (active, NPS ≥ 8)
├── What they get:
│   ├── Referrer: 1 tháng miễn phí / Credit / Cash reward
│   └── Referee: Discount 20% tháng đầu tiên
├── How to refer: Unique referral link + Email template
├── Tracking: UTM + Referral code in CRM
├── Terms: Referee must be new customer, activate within 30 days
```

## Execution
```
// turbo
├── Ask timing: After positive milestone (NPS survey ≥ 9, QBR success)
├── CSM: Personal ask during check-in
├── Automated: In-app prompt, email trigger
├── Track: Referrals sent → Converted → Revenue attributed
├── Report: Monthly referral metrics → Sales Manager
```
