# SOP-205: Social Media Management — Quản lý Truyền thông Xã hội

> **Actor**: Social Media Specialist + Marketing Manager | **Trigger**: Quản lý kênh social hàng ngày

## Content Calendar (Monthly)
```
// turbo
Actor: Social Media Specialist + Content Writer
├── Week 1: Plan content themes + key dates
├── Phân bổ:
│   ├── 40% Value (tips, education, how-to)
│   ├── 25% Product (features, use cases, releases)
│   ├── 20% Culture (team, behind-the-scenes, hiring)
│   └── 15% Engagement (polls, Q&A, trending topics)
├── Platform-specific:
│   ├── Facebook: 3-5 posts/week, mix carousel + video
│   ├── LinkedIn: 3-4 posts/week, thought leadership + case studies
│   ├── TikTok: 2-3 videos/week (if applicable)
│   └── YouTube: 1-2 videos/month (tutorial, webinar)
├── Approval: Marketing Manager review before scheduling
```

## Daily Operations
```
// turbo
├── Morning: Check notifications, respond to comments/DMs (< 2h SLA)
├── Midday: Publish scheduled content, monitor engagement
├── Afternoon: Community management, engage with relevant posts
├── End of day: Log metrics, flag issues
├── Crisis: Negative viral → Escalate to CMO → SOP-106
```

## Community Management Rules
```
// turbo
├── Response time: <2 hours business hours, <12 hours off-hours
├── Positive comment: Like + Thank + Follow-up question
├── Question: Answer helpfully or tag relevant person
├── Complaint: Acknowledge → Apologize → Move to DM → Resolve
├── Troll/Spam: Hide + Block (don't engage)
├── Crisis: Do NOT respond → Escalate to CMO immediately
```

## Monthly Reporting
```
// turbo
├── Metrics per platform: Followers, Reach, Engagement Rate, Clicks
├── Best/Worst performing posts (with WHY analysis)
├── Competitor activity summary
├── Next month recommendations
└── Template: weekly-report.md
```
