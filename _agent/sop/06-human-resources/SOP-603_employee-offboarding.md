# SOP-603: Employee Offboarding — Thủ tục Nghỉ việc

> **Actor**: HR Ops + Manager + IT | **Trigger**: Nhân viên nghỉ việc (resigned/terminated)

## Timeline

### Day 0: Notification
```
// turbo
├── Resignation: Nhận đơn → Manager + CHRO thông báo
├── Termination: CHRO + GC review legal basis → Formal notice
├── Notice period: 30-45 ngày (theo BLLĐ 2019)
├── Exit interview: CHRO hoặc HR Ops schedule (cuối notice period)
```

### During Notice Period
```
// turbo
├── Knowledge transfer plan:
│   ├── Document ongoing projects, passwords, processes
│   ├── Identify successor or temporary owner
│   ├── Handover meetings (2-3 sessions minimum)
│   └── Update relevant SOPs, documentation
├── Manager: Communicate to team (tone: respectful, positive)
├── HR Ops: Calculate final settlement (lương còn lại, phép chưa nghỉ, thưởng)
```

### Last Day Checklist
```
// turbo
├── [ ] IT: Revoke all accounts (email, Slack, GitHub, SaaS tools)
├── [ ] IT: Collect laptop, equipment
├── [ ] HR: Final settlement calculation → CFO approve
├── [ ] HR: BHXH closure (chốt sổ BHXH)
├── [ ] HR: Issue experience certificate (Giấy xác nhận)
├── [ ] Manager: Confirm knowledge transfer complete
├── [ ] Legal: NDA, non-compete reminders
├── [ ] HR: Exit interview conducted
├── [ ] Finance: Final payment processed (within 14 days per law)
```

### Post-Exit
```
// turbo
├── EXIT analysis: CHRO review patterns (why people leave)
├── If voluntary: Was it preventable? What can we learn?
├── If pattern: 3+ departures for same reason → Structural fix needed
├── Alumni network: Stay connected (LinkedIn, future referral)
```
