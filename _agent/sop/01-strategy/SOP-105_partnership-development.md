# SOP-105: Partnership Development — Phát triển Đối tác

> **Actor**: CEO + CSO + CMO | **Trigger**: Tìm kiếm, đánh giá, hoặc quản lý đối tác

## Quy trình 5 Bước

### 1. IDENTIFY — Xác định Đối tác Tiềm năng
```
// turbo
├── Partner types: Technology / Channel / Integration / Co-marketing / Strategic
├── Criteria: Market fit, Reach, Reputation, Complementary strengths
├── Longlist: 10-20 candidates → Shortlist: 3-5
```

### 2. EVALUATE — Đánh giá
```
// turbo
├── Strategic fit: Align với vision VCT?
├── Commercial value: Revenue opportunity? CAC reduction?
├── Risk: Reputation risk? Competitive risk? Dependency?
├── Effort: Integration cost? Maintenance? Exclusivity?
└── Score: 1-10 per criterion → Rank
```

### 3. ENGAGE — Tiếp cận
```
// turbo
├── Intro: Mutual connection preferred, cold outreach if needed
├── First meeting: Mutual value exploration (NOT pitch)
├── Pilot proposal: Small-scope collaboration to test fit
├── Legal: NDA trước khi share sensitive info → Contract Specialist
```

### 4. FORMALIZE — Chính thức hóa
```
// turbo
├── MOU/LOI: Outline terms, scope, timeline
├── Agreement: Contract Specialist draft partnership agreement
├── Legal review: General Counsel
├── Key terms: Revenue share, exclusivity, exit clause, IP ownership
├── Sign + Announce: CEO + CMO coordinate
```

### 5. MANAGE — Quản lý
```
// turbo
├── Partner Manager (CSO or dedicated person)
├── QBR with partner: Review metrics, plan next quarter
├── Metrics: Revenue from partnership, leads generated, brand lift
├── Renewal: Evaluate annually, evolve terms
├── Exit: Define criteria + process for ending partnership
```
