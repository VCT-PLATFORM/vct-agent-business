# SOP-106: Crisis Management — Xử lý Khủng hoảng

> **Actor**: CEO + General Counsel + CMO | **Trigger**: Khủng hoảng PR, bảo mật, pháp lý, hoặc vận hành

## Phân loại Khủng hoảng

| Level | Mô tả | Ví dụ | Response Time |
|-------|-------|-------|-------------|
| **L1 Critical** | Đe dọa sự tồn tại công ty | Data breach lớn, kiện tụng, scandal | <1 giờ |
| **L2 Major** | Ảnh hưởng nghiêm trọng uy tín/doanh thu | PR xấu viral, khách hàng lớn churn | <4 giờ |
| **L3 Moderate** | Ảnh hưởng cục bộ | Complaint nhỏ viral, bug gây mất data 1 user | <24 giờ |

## Quy trình Xử lý

### 1. ASSESS — Đánh giá (30 phút)
```
// turbo
Actor: CEO + Crisis Team
├── WHAT: Đã xảy ra chuyện gì? Facts only, no speculation
├── WHO: Ai bị ảnh hưởng? Customers, partners, employees?
├── HOW BAD: Scale 1-10? Trend tệ hơn hay ổn định?
├── LEGAL: Có vi phạm pháp luật? → General Counsel immediately
├── Form Crisis Team: CEO, GC, CMO, CTO, COO (as needed)
```

### 2. CONTAIN — Kiểm soát (2 giờ)
```
// turbo
├── Stop the bleeding: Immediate actions to prevent escalation
├── Internal comms: Align messaging (1 voice = CEO or spokesperson)
├── External hold: NO public comment until messaging approved
├── Preserve evidence: Screenshots, logs, communications
├── Legal: General Counsel advise on disclosure obligations
```

### 3. COMMUNICATE — Truyền thông (4 giờ)
```
// turbo
Actor: CMO + CEO
├── Internal first: ALL employees know before public
├── Stakeholder priority: Customers → Partners → Media → Public
├── Message framework:
│   ├── ACKNOWLEDGE: "Chúng tôi biết vấn đề..."
│   ├── TAKE RESPONSIBILITY: Không đổ lỗi
│   ├── ACTION: "Chúng tôi đang làm gì..."
│   ├── COMMITMENT: "Chúng tôi sẽ đảm bảo..."
│   └── CONTACT: Đầu mối liên hệ
├── Channels: Email direct → Website statement → Social media
├── Monitor: Track reactions, respond to questions
```

### 4. RESOLVE — Giải quyết
```
// turbo
├── Root cause: 5 Whys analysis
├── Fix: Technical, operational, policy changes
├── Compensation: Nếu customers affected (discount, service credit)
├── Legal resolution: If needed
├── Timeline: Set realistic milestones, communicate progress
```

### 5. RECOVER — Phục hồi
```
// turbo
├── Post-crisis review: What happened, what we did, what we learned
├── Process changes: Update SOPs to prevent recurrence
├── Rebuild trust: Proactive communication, transparency report
├── Monitor: Sentiment tracking 30/60/90 ngày
├── Document: Full crisis report → CEO memory/decisions/
```
