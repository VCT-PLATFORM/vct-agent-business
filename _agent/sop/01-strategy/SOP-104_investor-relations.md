# SOP-104: Investor Relations — Gọi vốn & Quan hệ Nhà đầu tư

> **Actor**: CEO + CFO + CSO | **Trigger**: Chuẩn bị gọi vốn hoặc báo cáo nhà đầu tư

## Quy trình Gọi vốn (Fundraising)

### Phase 1: PREPARE (T-8 tuần)
```
// turbo
Actor: CEO + CFO + CSO
├── Data Room chuẩn bị:
│   ├── Pitch Deck (15-20 slides)
│   ├── Financial Model (3-5 year projection, 3 scenarios)
│   ├── Cap Table
│   ├── Legal documents (Giấy ĐKKD, điều lệ, HĐLĐ key people)
│   ├── Product demo / screenshots
│   ├── Customer testimonials / case studies
│   └── Team bios
├── Valuation preparation: CFO model (DCF, Comparable, Precedent)
├── Target investor list: 30-50 investors, tiered A/B/C
```

### Phase 2: PITCH (T-4 tuần)
```
// turbo
├── Pitch Deck Structure (Sequoia format):
│   ├── 1. Problem (Why now?)
│   ├── 2. Solution (What you do)
│   ├── 3. Market (TAM/SAM/SOM)
│   ├── 4. Product (Demo/screenshots)
│   ├── 5. Traction (Metrics / Growth)
│   ├── 6. Business Model (Revenue)
│   ├── 7. Competition (Why you win)
│   ├── 8. Team (Why this team)
│   ├── 9. Financials (Key metrics)
│   └── 10. Ask (How much, use of funds)
├── Practice pitch: 5 lần trước khi gặp investor thật
├── Outreach sequence: Intro → Meet → Follow-up
```

### Phase 3: DUE DILIGENCE
```
// turbo
├── Financial DD: CFO + Accounting Manager chuẩn bị
├── Legal DD: General Counsel chuẩn bị
├── Technical DD: CTO chuẩn bị (architecture, security, scalability)
├── Customer DD: CSM chuẩn bị (references, NPS, retention data)
├── HR DD: CHRO (org chart, key person risk, ESOP plan)
```

### Phase 4: CLOSE
```
// turbo
├── Term Sheet negotiation: CEO + General Counsel
├── Key terms: Valuation, Liquidation preference, Board seat, Anti-dilution
├── Shareholder agreement: General Counsel review
├── Wire transfer + Cap table update
├── Announcement plan: CMO prepare comms
```
