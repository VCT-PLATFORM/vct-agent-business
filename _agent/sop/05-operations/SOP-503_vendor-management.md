# SOP-503: Vendor Management — Quản lý Nhà cung cấp

> **Actor**: COO + Accounting Manager | **Trigger**: Onboard, review, hoặc offboard vendor

## Vendor Lifecycle
```
// turbo
1. ASSESS: Evaluate vendor (trigger SOP-406 for tech tools)
2. CONTRACT: Draft agreement (trigger SOP-701)
3. ONBOARD: Setup access, payment terms, SLA agreement
4. MANAGE: Monthly/Quarterly review (performance vs SLA)
5. REVIEW: Annual renewal evaluation (renew / renegotiate / replace)
6. OFFBOARD: Data export, access revocation, contract termination
```

## Vendor Registry
```
// turbo
Maintain a central vendor list:
├── Vendor name + Contact person
├── Service provided
├── Annual cost
├── Contract start/end date
├── Payment terms (NET 15/30/45)
├── Renewal date (alert 60 days before)
├── Owner (who manages this vendor relationship)
└── Performance rating (last review)
```
