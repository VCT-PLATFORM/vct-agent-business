# SOP-504: Continuous Improvement — Cải tiến Liên tục

> **Actor**: COO + All Managers | **Frequency**: Ongoing + Bi-weekly Retro

## Retrospective Process (Bi-weekly)
```
// turbo
Format: Start / Stop / Continue (30 min)
├── START: What should we start doing? (new practices)
├── STOP: What should we stop doing? (waste, frustration)
├── CONTINUE: What's working well? (keep doing)
├── Vote: Each person votes top 1 per category
├── Action: Top-voted items → Owner + Deadline
├── Follow-up: Review previous retro actions at start
```

## Improvement Tracking
```
// turbo
├── Kaizen Board:
│   ├── IDEA: New improvement suggestions
│   ├── EVALUATE: Being assessed (impact vs effort)
│   ├── IN PROGRESS: Being implemented
│   ├── DONE: Implemented, monitoring impact
│   └── MEASURED: Impact confirmed (with data)
├── Each improvement: Before metric → After metric → Delta
├── Monthly: Share top 3 improvements company-wide
```

## Root Cause Analysis (When needed)
```
// turbo
├── 5 Whys: Why? → Why? → Why? → Why? → Why? → Root cause
├── Fishbone Diagram: Categories → Causes → Sub-causes
│   ├── People: Skill gap? Workload? Communication?
│   ├── Process: Missing step? Unclear? Outdated?
│   ├── Technology: Bug? Limitation? Wrong tool?
│   ├── Data: Inaccurate? Missing? Late?
│   └── External: Market change? Regulation? Vendor?
├── Action: Fix root cause, NOT symptom
```
