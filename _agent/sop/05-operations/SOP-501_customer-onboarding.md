# SOP-501: Customer Onboarding — Hội nhập Khách hàng Mới

> **Actor**: CSM + Support Lead | **Trigger**: New customer contract signed

## Onboarding Timeline

| Day | Milestone | Owner | Deliverable |
|-----|----------|-------|-----------|
| 0 | Handoff from Sales → CSM | AE + CSM | Handoff brief |
| 1 | Welcome email + Account setup | CSM + Support | Credentials, welcome guide |
| 1-3 | Kickoff call (30 min) | CSM | Success plan, key milestones |
| 3-7 | Technical setup / Data migration | Support + Tech | Environment ready |
| 7-14 | Training sessions (2-3 sessions) | CSM | Recording + Materials |
| 14 | First Value Milestone | CSM | User achieves first "aha!" moment |
| 30 | Check-in call | CSM | Usage review, feedback, adjustments |
| 60 | Health check | CSM | Feature adoption, satisfaction survey |
| 90 | QBR #1 + Success confirmed | CSM | ROI report, expansion discussion |

## Success Criteria
```
// turbo
├── Time-to-Value: <14 days (user completes their first key workflow)
├── Adoption: >3 features actively used by Day 30
├── Stickiness: >2 users active by Day 30
├── Satisfaction: CSAT ≥ 8 at Day 30 check-in
└── Churn risk: If any criteria missed → Escalate to CSM Manager
```
