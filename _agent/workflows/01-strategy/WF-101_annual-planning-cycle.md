# WF-101: Annual Planning Cycle

> Workflow: **CEO → CSO → All C-suite → Depts → Board**

```
Oct                Nov                Dec              Jan
Review Year → Analyze Env → Set Strategy → Cascade OKRs → Execute
  All Depts    CSO+Data     CEO+CSO+CFO    C-suite→Mgrs    All
```

## Quarterly Breakdown
```
// turbo
Q4 of Current Year:
├── Oct: Year-in-review (SOP-103 Business Review)
│   ├── Each dept presents Annual Report
│   ├── Data team prepares trend analysis
│   └── Finance presents Year-end forecast
├── Nov: Strategic analysis (SOP-101 Strategic Planning)
│   ├── PESTEL + SWOT + Porter's refresh
│   ├── Competitive landscape update
│   ├── Scenario planning (3 scenarios)
│   └── CEO + CSO draft strategic options
├── Dec: Strategy decision
│   ├── Board input/approval
│   ├── Budget allocation → CFO models
│   ├── Company OKRs set (SOP-102 OKR)
│   └── Headcount plan → CHRO
├── Jan (Q1 next year): Cascade & execute
│   ├── Dept OKRs aligned to Company OKRs
│   ├── Team OKRs aligned to Dept OKRs
│   ├── All hands: CEO presents Annual Plan
│   └── Quarterly execution begins
```
