# WF-106: Market Entry

> Workflow: **CSO → Data → CMO → CFO → CEO**

```
Research → Validate → Strategy → Plan → Execute → Measure
 CSO+Data   AE+PM     CSO+CEO   All    Mktg+Sales  Data
```

## Flow
```
// turbo
1. RESEARCH (CSO + Data Analyst):
   ├── Market sizing: TAM → SAM → SOM
   ├── Customer discovery: 10-20 interviews
   ├── Competitive landscape mapping
   ├── Regulatory requirements (GC input)
   └── SOP-101 Step 2 (Multi-framework analysis)

2. VALIDATE:
   ├── Pilot: 5-10 target customers → Run sales process
   ├── Product-market fit signals: Willingness to pay? Repeat usage?
   ├── Localization needs: Language? Payment? Compliance?
   └── Go/No-Go: CEO decision (SOP-002 Multi-Agent Debate)

3. PLAN (if GO):
   ├── GTM strategy: Direct/Channel/PLG/Hybrid
   ├── Budget: CFO allocate (SOP-301)
   ├── Hiring: Local team? Remote? Partner? (SOP-601)
   ├── Legal: Entity setup, regulations (GC)
   └── Timeline: 6-12 month plan

4. EXECUTE:
   ├── Marketing localization (SOP-201, SOP-202)
   ├── Sales activation (SOP-203)
   ├── Customer success setup (SOP-501)
   └── Support localization

5. MEASURE (monthly for first 6 months):
   ├── Acquisition: Leads, Trials, Customers
   ├── Revenue: MRR, Pipeline
   ├── Unit economics: CAC, LTV in new market
   └── Decision @ Month 6: Scale / Pivot / Exit
```
