# WF-105: Fundraising Campaign

> Workflow: **CEO → CFO → CSO → CTO → GC → CMO**

```
Prepare (8w) → Target (4w) → Pitch (4w) → DD (4w) → Close (2w) → Announce
 CEO+CFO       CEO         CEO+CFO      All        CEO+GC      CMO
```

## Flow
```
// turbo
1. PREPARE: Data room + Financial model + Pitch deck
   ├── CFO: 3-5 year model, metrics package
   ├── CSO: Market analysis, competitive moat
   ├── CTO: Technical architecture, scalability proof
   └── SOP-104 (Investor Relations)

2. TARGET: 30-50 investor list, tiered A/B/C
   ├── Research: Fund thesis match, stage fit, portfolio conflicts
   └── Warm intros preferred (2x higher conversion)

3. PITCH: Pipeline management (Notion/CRM)
   ├── First meeting: 30 min, story-driven
   ├── Follow-up: Data room access
   ├── Partner meeting: Deep dive
   └── Track: Meeting → Follow-up → DD → Term sheet

4. DUE DILIGENCE: All depts prepare docs
   ├── Financial DD → CFO + Accounting
   ├── Legal DD → GC
   ├── Technical DD → CTO
   └── Customer DD → CSM (references)

5. CLOSE: Term sheet → SHA → Wire → Cap table update
   └── GC reviews all legal docs

6. ANNOUNCE: Press release + Team comms + Customer comms
   └── CMO coordinates messaging
```
