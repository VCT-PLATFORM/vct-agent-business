# WF-107: Crisis Response

> Workflow: **CEO → GC → CMO → All C-suite** | Tham chiếu: SOP-106

```
Detect → Assess → Contain → Communicate → Resolve → Recover
 Anyone   CEO+GC   Crisis    CMO+CEO       All       All
              Team
```

## Activation
```
// turbo
WHEN: PR crisis, security breach, legal threat, key-person loss, or any event
      threatening company reputation/operations/survival
WHO activates: ANYONE can trigger → CEO makes L1/L2/L3 classification
├── L1 Critical (<1h): All C-suite mobilized
├── L2 Major (<4h): CEO + relevant C-suite
└── L3 Moderate (<24h): Relevant dept head
```

## War Room Roles
```
// turbo
├── Commander: CEO (final decisions)
├── Legal Advisor: General Counsel (what we CAN/CANNOT say/do)
├── Spokesperson: CEO or CMO (ONE voice only)
├── Technical Lead: CTO (if tech-related)
├── Ops Lead: COO (business continuity)
└── Scribe: Record all decisions + actions + timeline
```
