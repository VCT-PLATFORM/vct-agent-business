# WF-703: Performance Cycle

> Workflow: **CHRO → Managers → Employees → Calibration Committee**

```
Q1 Start: Set Goals → Weekly 1:1 → Q End: Review → Calibrate → Reward
  Mgr+Emp              Mgr+Emp       Mgr+Emp      CHRO+Mgrs   CFO+CHRO
```

## Flow → Xem SOP-602 (Performance Review) cho chi tiết

## Annual Cadence
```
// turbo
January: Goal setting for the year (aligned to Company OKRs)
├── Each employee: 3-5 goals linked to team/dept OKRs
└── Manager approves, CHRO reviews

Quarterly: Formal review conversation
├── Q1 (March): First check-in
├── Q2 (June): Mid-year review (MOST IMPORTANT — course correct)
├── Q3 (September): Pre-year-end check
└── Q4 (December): Year-end review + Rating

December: Annual calibration
├── Managers present team ratings → Dept Head + CHRO
├── Ensure fairness across teams
├── Rating distribution sanity check
└── Output: Final ratings → Compensation review

January (next year): Rewards
├── Salary adjustment (based on rating + market data)
├── Bonus payout
├── Promotion announcements
├── ESOP grants (if applicable)
└── New cycle begins
```
