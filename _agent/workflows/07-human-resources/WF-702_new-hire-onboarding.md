# WF-702: New Hire Onboarding

> Workflow: **HR Ops → Manager → L&D → Buddy → IT**

```
Pre-Day1 → Day 1 → Week 1 → Day 30 → Day 60 → Day 90
  HR+IT    HR+Mgr   Mgr+L&D   Mgr     Mgr     Mgr+CHRO
```

## Flow → Adapts SOP-601 Step 7

## Detailed Timeline
```
// turbo
PRE-DAY 1 (HR Ops + IT):
├── Equipment: Laptop, monitor, peripherals ordered
├── Accounts: Email, Slack, GitHub, tools provisioned
├── Workspace: Desk/setup ready (or remote kit shipped)
├── Welcome kit: Swag, handbook, welcome letter from CEO
├── Buddy assigned: Peer mentor for first 90 days
└── Calendar: First-week meetings pre-scheduled

DAY 1 (HR + Manager):
├── Morning: HR welcome, paperwork, benefits enrollment
├── Late AM: Manager welcome, team intro, role expectations
├── Afternoon: IT setup, tool walkthroughs, Slack channels
├── End of day: Buddy coffee chat + Q&A

WEEK 1 (Manager + L&D):
├── Product deep-dive (2-3 sessions)
├── COMPANY_CONTEXT.md walkthrough
├── Team meetings attendance (listen & learn)
├── First small task assigned
├── End-of-week check-in with Manager

DAY 30 CHECK-IN (Manager):
├── How are you feeling? (wellbeing)
├── What's clear? What's confusing?
├── First deliverable reviewed
├── OKRs set for probation period
└── Feedback: Both directions

DAY 60 CHECK-IN (Manager):
├── Progress vs OKRs
├── Integration with team
├── Development areas identified
└── IDP started (SOP-604)

DAY 90 PROBATION REVIEW (Manager + CHRO):
├── Performance vs expectations
├── Culture fit assessment
├── Decision: Confirm / Extend probation / Terminate
├── If confirmed: Full onboarding complete 🎉
└── Set first full-quarter OKRs
```
