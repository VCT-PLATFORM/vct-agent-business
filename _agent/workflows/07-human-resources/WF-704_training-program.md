# WF-704: Training Program

> Workflow: **L&D Specialist → Managers → External trainers**

```
Assess → Design → Deliver → Evaluate → Iterate
 L&D+Mgr   L&D     L&D/Ext    L&D       L&D
```

## Flow → Xem SOP-604 (Learning & Development) cho IDP details

## Program Types & Flow
```
// turbo
1. ASSESS (L&D + Managers):
   ├── Skills gap analysis (per team/individual)
   ├── Company needs: New tools, processes, tech stack
   ├── Career paths: What skills needed for next level?
   └── Budget: Annual training budget allocation

2. DESIGN (L&D):
   ├── Internal workshop: Subject matter expert teaches
   ├── External course: Udemy, Coursera, LinkedIn Learning
   ├── Coaching: 1-on-1 with external coach (leadership)
   └── Conference: Industry events + knowledge sharing

3. DELIVER:
   ├── Schedule: Avoid peak productivity hours
   ├── Materials: Recording + Slides + Resources
   └── Hands-on: Practice exercises, not just lectures

4. EVALUATE (Kirkpatrick Model):
   ├── Level 1 Reaction: Post-training survey (did they like it?)
   ├── Level 2 Learning: Quiz/assessment (did they learn?)
   ├── Level 3 Behavior: 30-day follow-up (are they applying it?)
   └── Level 4 Results: Business impact (did it improve KPIs?)
```
