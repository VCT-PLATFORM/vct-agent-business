# WF-705: Exit Process

> Workflow: **Employee → Manager → HR Ops → IT → CHRO**

## Flow → Xem SOP-603 (Employee Offboarding) cho checklists

```
// turbo
Trigger: Employee resigns or Termination decision made

Day 0: Notification received / Given
├── Manager + CHRO informed
├── Counter-offer decision (if voluntary + key person)
├── Notice period starts (30-45 days per BLLĐ)

Day 1-25: Knowledge Transfer
├── Manager creates transfer plan
├── Successor identified (or interim owner)
├── 2-3 handover sessions
├── Documentation updated

Day 25-30: Administrative
├── HR: Final settlement calc
├── HR: BHXH closure
├── IT: Deprovisioning scheduled
├── Legal: NDA reminders

Last Day:
├── Exit interview (CHRO or HR Ops)
├── Equipment return
├── Access revoked
├── Farewell

Post-Exit:
├── Final payment within 14 days
├── CHRO: Pattern analysis (voluntary attrition reasons)
├── If 3+ exits same reason → Structural intervention
```
