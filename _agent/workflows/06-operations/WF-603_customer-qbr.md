# WF-603: QBR (Quarterly Business Review) with Customer

> Workflow: **CSM → Data Analyst → AE → Customer**

```
T-2w Prep → T-1w Build Deck → D-Day Present → D+1 Actions → D+7 Follow-up
 CSM+Data     CSM               CSM+AE          CSM           CSM
```

## Flow
```
// turbo
1. PREP (T-2 weeks):
   ├── Data Analyst pulls: Usage metrics, adoption trends, support tickets
   ├── CSM identifies: Wins, challenges, expansion opportunities
   ├── AE identifies: Upsell/cross-sell angles
   └── Review: Previous QBR action items — completed?

2. BUILD DECK:
   ├── Slide 1: Executive summary (health score, key facts)
   ├── Slide 2: Usage dashboard (adoption, active users, features)
   ├── Slide 3: Support summary (tickets, resolution, CSAT)
   ├── Slide 4: Value delivered (ROI, goals achieved)
   ├── Slide 5: Roadmap preview (upcoming features relevant to them)
   ├── Slide 6: Growth discussion (what else can VCT help with?)
   └── Slide 7: Action items (both sides)

3. PRESENT (45-60 min):
   ├── Invite: Customer champion + decision maker
   ├── CSM presents Slides 1-5
   ├── Open discussion on Slide 6
   ├── Agree on action items Slide 7
   └── If expansion interest → Invite AE to next meeting

4. FOLLOW-UP:
   ├── Send QBR summary + deck (within 24h)
   ├── Action items tracked in CRM
   └── Lưu vào: operations-pm/customer-success-manager/memory/deliverables/
```
