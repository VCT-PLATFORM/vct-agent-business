# WF-005: Hire to Perform — Hành trình Nhân sự

> Workflow: **HR → Manager → L&D → Finance**

```
Request → Source → Interview → Offer → Onboard → Develop → Review → Exit
 Manager  Recruiter  MgrHR      HR     HR+Mgr    L&D     Manager   HR
```

## Flow
```
// turbo
1. REQUEST: Manager identifies need → CHRO + CFO approve headcount
   └── SOP-601 Step 1

2. SOURCE & HIRE: JD → Sourcing → Interview (4 rounds) → Offer
   └── SOP-601 Steps 2-6

3. ONBOARD (30/60/90): Equipment → Training → Buddy → Milestones
   └── SOP-601 Step 7 + SOP-501 (adapted for employees)

4. DEVELOP: IDP → Training budget → 70-20-10 learning
   └── SOP-604 (Learning & Development)

5. PERFORM: Weekly 1-on-1 → Quarterly review → Annual calibration
   └── SOP-602 (Performance Review)

6. GROW or EXIT:
   ├── Promotion: 2+ quarters Rating 4-5 → New role → New OKRs
   ├── Lateral move: Cross-functional rotation
   └── Exit: SOP-603 (Offboarding) → Knowledge transfer → Exit interview
```
