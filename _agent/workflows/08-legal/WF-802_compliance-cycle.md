# WF-802: Compliance Cycle

> Workflow: **Compliance Officer → GC → All Depts → CEO**

```
Q1 Assess → Q1 Plan → Q2-Q3 Implement → Q4 Audit → Report
  CO+GC       CO         All Depts        CO+GC     CO→CEO
```

## Annual Flow
```
// turbo
Q1: ASSESS & PLAN
├── Compliance Officer: Regulatory landscape scan
│   ├── New laws/regulations passed?
│   ├── Existing compliance gaps?
│   └── Industry standards evolving? (SOC2, ISO 27001)
├── Risk assessment: Top 10 compliance risks ranked
├── Plan: Actions to close gaps → Owners → Deadlines
└── Budget: Any external counsel/audit needed?

Q2-Q3: IMPLEMENT
├── Each dept implements assigned compliance actions
├── Training: Compliance awareness for all employees
├── Documentation: Policies updated, procedures documented
├── Systems: Technical controls implemented (encryption, access, logging)

Q4: AUDIT & REPORT
├── Quarterly compliance audit → SOP-702
├── Score: Overall compliance posture (%)
├── Report: CEO + Board
├── External: Prepare for external audit if needed (SOP-306)
```
