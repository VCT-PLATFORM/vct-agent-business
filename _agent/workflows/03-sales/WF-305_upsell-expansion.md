# WF-305: Upsell & Expansion

> Workflow: **CSM → AE → Product → Finance**

```
Identify Opportunity → Qualify → Propose → Close → Expand Account
      CSM                AE       AE+PM     AE      Finance
```

## Expansion Signals
```
// turbo
├── Usage: Feature usage >80% of limit
├── Growth: Customer adding more users/seats
├── Satisfaction: NPS 9-10, CSAT consistently high
├── Request: Customer asks about premium features
├── Business: Customer raised funding, grew team
└── QBR: During quarterly review, needs emerged
```

## Flow
```
// turbo
1. CSM identifies expansion signal → Alert AE
2. AE + CSM joint call with customer:
   ├── Explore additional needs (consultative, not pushy)
   ├── Demo premium features relevant to their use case
   └── Discuss pricing/ROI
3. If interested → Upsell proposal (simplified, amendment to existing contract)
4. Contract amendment → Contract Specialist (simplified process)
5. Invoice update → Accounting Manager (MRR increase)
6. Metrics track: Expansion MRR, NRR (Net Revenue Retention)
```
