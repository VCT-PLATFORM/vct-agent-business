# /content-creation — Quy trình Tạo Nội dung

> **SOP-006** | Áp dụng khi: Viết bài đăng, email, blog, ad copy, hoặc bất kỳ nội dung marketing nào
> Actor: Marketing Manager → Content Writer + SEO + Designer + Social Media

---

## Khi nào sử dụng
- "Viết bài đăng Facebook/LinkedIn/Blog"
- "Tạo content cho chiến dịch"
- "Viết email marketing"
- Bất kỳ yêu cầu tạo nội dung nào

## Quy trình 8 Bước

### Bước 1: BRIEF — Tạo Content Brief
```
// turbo
Actor: Marketing Manager (hoặc người yêu cầu)
Output:
├── Objective: Mục tiêu nội dung (awareness? leads? education?)
├── Audience: Viết cho ai? (ICP từ COMPANY_CONTEXT.md)
├── Channel: Facebook / LinkedIn / Blog / Email / Ad?
├── Key Message: 1 thông điệp cốt lõi
├── CTA: Hành động mong muốn từ người đọc
├── Tone: Theo Brand Voice (xem Brand Manager SKILL.md)
├── References: Link, data, ví dụ tham khảo
├── Deadline: Khi nào cần
```

### Bước 2: SEO RESEARCH (Nếu Blog/Website)
```
// turbo
Actor: SEO Specialist
├── Keyword research → Primary keyword + 3-5 secondary
├── Search intent analysis (Informational / Commercial / Transactional)
├── Competitor content audit (Top 3 ranking articles)
├── Content brief: H2 headings gợi ý, word count target, internal links
Output: SEO Brief → Content Writer
```

### Bước 3: WRITE — Viết Nội dung
```
// turbo
Actor: Content Writer
├── Đọc brief + SEO brief (nếu có)
├── Viết theo framework phù hợp:
│   ├── Social: Hook → Value → CTA
│   ├── Blog: AIDA hoặc PAS
│   ├── Email: Subject → Preview → Body → P.S.
│   └── Ad: Headline → Description → CTA
├── Self-check:
│   ├── ✅ Hook mạnh trong câu đầu?
│   ├── ✅ Không placeholder?
│   ├── ✅ CTA rõ ràng?
│   ├── ✅ Đúng word count?
│   └── ✅ Brand voice?
Output: Draft v1
```

### Bước 4: DESIGN — Thiết kế Visual
```
// turbo
Actor: Graphic Designer
├── Đọc content draft → Xác định visual concept
├── Tạo design brief (composition, colors, typography)
├── Đúng dimensions cho platform (xem Designer SKILL.md)
├── Text trên ảnh khớp với content
├── Tạo AI Image Prompt nếu cần generate ảnh
Output: Visual asset description / prompt
```

### Bước 5: OPTIMIZE — Tối ưu Platform
```
// turbo
Actor: Social Media Specialist (nếu social) hoặc SEO (nếu blog)
├── Social:
│   ├── Hashtags (5-10, mix popular + niche)
│   ├── Best posting time (xem SKILL.md)
│   ├── Seeding strategy
│   └── Engagement plan (reply protocol)
├── Blog:
│   ├── Meta title + description
│   ├── Schema markup suggestion
│   ├── Internal/External links
│   └── Featured image alt text
Output: Optimization layer added to content
```

### Bước 6: REVIEW — Manager Review
```
// turbo
Actor: Marketing Manager
├── Content quality: Compelling? Clear? Actionable?
├── Brand consistency: Tone, visual, messaging (check with Brand Manager)
├── SEO compliance: Keywords natural? Structure correct?
├── Cross-check: Text + Visual + CTA = Coherent message
├── PASS → Bước 7
└── FAIL → Feedback → Quay lại Writer/Designer
```

### Bước 7: APPROVE — Executive Approval
```
// turbo
Actor: CMO (nếu campaign) hoặc Marketing Manager (nếu routine)
├── Strategic alignment check
├── Final proofread
├── APPROVE → Publish
```

### Bước 8: PUBLISH & MEASURE
```
// turbo
Actor: Social Media Specialist / Content Writer
├── Publish content
├── Execute seeding strategy (nếu social)
├── Track metrics:
│   ├── Social: Reach, Engagement, Clicks, Shares
│   ├── Blog: Traffic, Time on page, Bounce rate, Rankings
│   ├── Email: Open rate, CTR, Unsubscribe
│   └── Ad: Impressions, Clicks, Conversions, CPA
├── Lưu content + metrics → memory/deliverables/
```
