# WF-203: Brand Refresh

> Workflow: **CMO → Brand Manager → Designer → Content → Legal**

```
Audit → Strategy → Design → Apply → Launch → Monitor
 Brand   CMO      Designer  All     CMO+Mktg  Brand
```

## Flow
```
// turbo
1. AUDIT (Brand Manager): Current brand health assessment
   ├── Visual audit: Is current identity outdated?
   ├── Voice audit: Tone consistent across channels?
   ├── Perception audit: Customer surveys, NPS verbatims
   └── Competitive audit: How competitors look/sound

2. STRATEGY (CMO + CEO):
   ├── Why refresh? (evolution, pivot, acquisition, outdated)
   ├── What changes: Visual / Voice / Both / Full rebrand
   ├── Brand architecture: If multi-product, how brands relate
   └── Budget & timeline

3. DESIGN (UX/UI Designer + Brand Manager):
   ├── Logo evolution / redesign
   ├── Color palette update
   ├── Typography refresh
   ├── Imagery/illustration style
   ├── Brand guidelines document update
   └── Trademark check → IP Specialist (SOP-704)

4. APPLY (All teams):
   ├── Website update
   ├── Social profiles update
   ├── Email templates update
   ├── Sales collateral update
   ├── Product UI update
   └── Internal templates update

5. LAUNCH (CMO):
   ├── Internal: All Hands announcement + Brand training
   ├── External: Blog post + Social announcement
   └── SOP-202 (Campaign Launch) for major rebrands

6. MONITOR (Brand Manager):
   ├── Adoption tracking: All materials updated?
   ├── Sentiment: Customer reaction
   └── Compliance: No old branding leaks
```
