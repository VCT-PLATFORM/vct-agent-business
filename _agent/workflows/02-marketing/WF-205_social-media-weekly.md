# WF-205: Social Media Weekly

> Workflow: **Social Media Specialist daily loop → Marketing Manager weekly review**

```
Monday        Tuesday-Thursday       Friday
Plan Week → Create + Publish → Engage → Report
 SMSpec       SMSpec+Writer      SMSpec   SMSpec→MktgMgr
```

## Weekly Rhythm
```
// turbo
Monday:
├── Review last week metrics (10 min)
├── Plan this week's posts (map to content calendar)
├── Draft/Schedule posts for Tue-Thu
├── Respond to weekend comments/DMs

Tuesday-Thursday:
├── 9AM: Publish scheduled post
├── 11AM: Engage with relevant industry posts
├── 2PM: Check comments/DMs → Respond
├── 4PM: Monitor trending topics → Reactive content if relevant
├── Create next day's content if not pre-scheduled

Friday:
├── Publish 1 post (lighter, engaging: poll/question/behind-scenes)
├── Weekly metrics report → Marketing Manager
├── Content performance analysis (best/worst posts)
├── Next week content calendar prep
```
