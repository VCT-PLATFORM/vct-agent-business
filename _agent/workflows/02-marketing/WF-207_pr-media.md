# WF-207: PR & Media

> Workflow: **CMO → Content Writer → CEO → Media**

## When to PR
```
// turbo
├── 🟢 Do PR: Major product launch, Funding round, Awards, Partnership
├── 🟡 Maybe: Milestone (100 customers), Hires (C-suite)
└── 🔴 Don't: Minor features, Self-congratulatory announcements
```

## Press Release Flow
```
// turbo
1. DECIDE: CMO evaluates newsworthiness → CEO approves
2. WRITE: Content Writer drafts press release
   ├── Headline (compelling, not clickbait)
   ├── Subhead (context)
   ├── Lead paragraph (who/what/when/where/why)
   ├── Quote from CEO/relevant executive
   ├── Supporting details + Data
   ├── Boilerplate (company description)
   └── Contact info
3. REVIEW: CMO → CEO → GC (legal check)
4. DISTRIBUTE:
   ├── Exclusive first (if strategic: 1 journalist, 24h head start)
   ├── Wire service (broader distribution)
   ├── Direct pitches to relevant journalists
   └── Syndicate on company blog + social
5. AMPLIFY: Social media, email newsletter, partner shares
6. MEASURE: Impressions, Pickups, Sentiment, Referral traffic
```

## Media Relationships
```
// turbo
├── Build: Maintain journalist contact list (beat, outlet, relationship)
├── Nurture: Share insights/data even when not asking for coverage
├── Respond: Media inquiries within 2 hours → CMO or CEO only
├── Crisis media: ONLY through designated spokesperson (SOP-106)
```
