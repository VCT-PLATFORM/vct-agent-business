# /campaign-launch — Quy trình Ra mắt Chiến dịch / Tính năng

> **SOP-007** | Áp dụng khi: Ra mắt sản phẩm, tính năng, chiến dịch marketing
> Actor: CMO + Product Manager + Cross-function

---

## Khi nào sử dụng
- Ra mắt tính năng sản phẩm mới
- Chiến dịch marketing lớn (seasonal, product launch)
- Rebranding hoặc major announcement
- Partnership launch

## Quy trình 5 Phase

### Phase 1: PLAN (T-4 tuần)
```
// turbo
Actor: CMO + Product Manager
├── Define launch tier:
│   ├── Tier 1 (Major): New product, major feature → Full GTM
│   ├── Tier 2 (Medium): Nice feature, improvement → Blog + Email + Social
│   └── Tier 3 (Minor): Bug fix, small update → Changelog only
├── Go-to-Market Brief:
│   ├── What: Sản phẩm/tính năng gì?
│   ├── Who: Target audience (segment cụ thể)
│   ├── Why: Key value proposition (1 sentence)
│   ├── When: Launch date
│   ├── How: Channels, content assets needed
│   └── Metrics: How we measure success
├── Budget request → CFO approve
├── Resource allocation → Assign owners per asset
```

### Phase 2: CREATE (T-3 đến T-1 tuần)
```
// turbo
Actor: Marketing Manager phân công

Asset Checklist (Tier 1):
├── [ ] Landing page copy → Content Writer
├── [ ] Hero image / Video → Graphic Designer
├── [ ] Blog post (deep dive) → Content Writer + SEO
├── [ ] Social posts (3-5 per platform) → Social Media + Content Writer
├── [ ] Email sequence (announcement + follow-up) → Content Writer
├── [ ] Sales collateral (one-pager, FAQ) → Account Executive + Content Writer
├── [ ] Internal comms → Marketing Manager
├── [ ] Press release (if newsworthy) → CMO
├── [ ] Customer success enablement → CSM materials
└── [ ] In-app notification copy → Product Manager + Content Writer
```

### Phase 3: REVIEW (T-1 tuần)
```
// turbo
Actor: Brand Manager + CMO
├── All assets reviewed for:
│   ├── Brand consistency (voice, visual, messaging)
│   ├── Accuracy (features, pricing, dates)
│   ├── Legal compliance (disclaimers, T&C) → General Counsel if needed
│   └── Cross-asset coherence (same story across channels)
├── Dry run: Preview all assets as end-user would see them
├── Sign-off: CMO approved ✅
```

### Phase 4: LAUNCH (D-Day)
```
// turbo
Actor: All Marketing team

Launch Sequence:
├── T-2h: Final checks (links, tracking, feature flags)
├── T-0: Publish all assets simultaneously:
│   ├── Website/Landing page live
│   ├── Blog post published
│   ├── Email blast sent
│   ├── Social posts scheduled/posted
│   ├── In-app notification triggered
│   └── Sales team notified
├── T+1h: Monitor initial metrics, fix any issues
├── T+4h: First engagement report
├── T+24h: Day-1 results summary
```

### Phase 5: MEASURE (D+1 đến D+30)
```
// turbo
Actor: Growth Analyst + Marketing Manager

Measurement Schedule:
├── Day 1: Reach, impressions, initial engagement
├── Week 1: Traffic, signups, activation from launch
├── Week 2-3: Conversion funnel analysis
├── Week 4: Full launch retro:
│   ├── What worked? (Repeat)
│   ├── What didn't? (Improve)
│   ├── Surprises? (Learn)
│   └── ROI calculation

Output: Launch Report → memory/deliverables/
Next: Input for future launches
```
