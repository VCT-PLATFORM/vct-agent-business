# Shared Knowledge — Tri thức Chung Toàn Công ty

> **Mục đích**: Lưu trữ tài liệu, thông tin, dữ liệu mà **TẤT CẢ phòng ban** cần tham chiếu.

## Cấu trúc

| Thư mục | Nội dung | Ví dụ |
|---------|---------|-------|
| `brand/` | Brand guidelines, logo, fonts, tone of voice | `brand-book.pdf`, `logo-kit.zip` |
| `product/` | Product specs, feature list, roadmap | `product-roadmap-2026.md` |
| `market/` | Nghiên cứu thị trường, competitive analysis | `competitive-landscape-q1-2026.md` |
| `policies/` | Chính sách nội bộ, nội quy, tài chính | `financial-policy.md` |
| `onboarding/` | Tài liệu cho nhân viên mới | `welcome-guide.md` |
| `emails/` | Lưu trữ email quan trọng | |
| `invoices/` | Hóa đơn đầu vào | |
| `marketing/` | Tài liệu marketing campaigns | |

## ⚠️ QUAN TRỌNG — Thư mục PHÁP LÝ đã chuyển

> Toàn bộ văn bản pháp luật đã được **hợp nhất** và chuẩn hóa tên sang:
>
> **`_agent/knowledge/legal/`**
>
> Thư mục `shared_knowledge/legal/` giữ lại bản gốc để tham chiếu.
> Tất cả agents phải đọc pháp lý từ `_agent/knowledge/legal/`.

## Cách sử dụng
- Agent sẽ tự động tham chiếu các file trong thư mục này cho context chung.
- Cập nhật thường xuyên để agent luôn có thông tin mới nhất.
- Đặt tên file rõ ràng, có ngày nếu là tài liệu định kỳ.
