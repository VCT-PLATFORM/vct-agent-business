# Templates Library — VCT Platform

Bộ sưu tập **mẫu tài liệu chuẩn** cho toàn bộ tổ chức. Mọi deliverable phải tuân theo format trong thư mục này để đảm bảo tính nhất quán và chuyên nghiệp.

## Danh sách Templates

| Template | File | Dùng bởi |
|----------|------|---------|
| Meeting Notes | `meeting-notes.md` | Tất cả |
| Weekly Report | `weekly-report.md` | Managers |
| Monthly Report | `monthly-report.md` | Executives |
| Proposal / Đề xuất | `proposal.md` | Tất cả |
| OKR Template | `okr.md` | Tất cả |
| Post-Mortem | `post-mortem.md` | Tech, Ops |
| Decision Record (ADR) | `decision-record.md` | Tất cả |
